# FinAPI(Financial API)

## Overview 
This is a Financial API project is developed in one study of backend applications that I'm working on.

The goal here is to build one api to handle Financial spents.

## Installing & Running
I'm using **nodejs** and **yarn**, but you can run with **npm** as well
just run:
```shell
yarn
```
or:
```shell
npm install
```

then to run the server
```shell
yarn dev
```

server will be listening to port 3333
